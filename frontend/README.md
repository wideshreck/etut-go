# Etut Frontend

Next.js frontend for the Etut education platform.

## Setup

```bash
pnpm install
cp .env.example .env.local
```

## Development

```bash
pnpm dev
```

## Code Quality

```bash
pnpm lint            # ESLint
pnpm format          # Prettier format
pnpm format:check    # Prettier check
pnpm build           # Type check + build
```
